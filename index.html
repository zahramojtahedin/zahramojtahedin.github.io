Optimization of ReLU-Based Layer Architectures in Deep Neural Networks Using Approximation Theory

A theory-driven study of how depth, width, connectivity, and sparsity influence the approximation capability of ReLU networks, with attention to architecture choices that remain useful in practice. Supervised by Prof. Siavash Bayat.

Approximation Theory ReLU Networks Network Complexity

Detection of Human Concentration Using Eyelid Movements and Neural Signals

A non-invasive concentration-monitoring framework combining eye-related measurements, neural signals, embedded sensing, and learning-based analysis to estimate cognitive attention. Supervised by Prof. Saeed Bagheri Shouraki.

EEG Eye Tracking Machine Learning

An FPGA-oriented CNN accelerator using residue-number arithmetic, divider-free modular reduction, quantized computation, and implementation-aware optimization for efficient inference.

FPGA RNS CNN Acceleration Quantization

A tile-structured FPGA architecture targeting transform processing and key-switching bottlenecks in TFHE bootstrapping, with emphasis on data locality, parallel butterfly processing, and efficient memory access.
